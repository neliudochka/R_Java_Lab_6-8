Номер залікової книжки: 1209
- С2 = 1209 % 2 = 1 => **Інтерфейс**: Set
- C3 = 1209 % 3 = 0 => **Внутрішня структуру колекції:** Масив із початковою кількістю елементів 15 та збільшенням кількості елементів на 30%

**Завдання**
